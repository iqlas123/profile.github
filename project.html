PROJECTS
VEHICLE TRACKING SYSTEM & THEFT DETECTION USING IOT The aim of this project is to develop a secure and reliable vehicle anti-theft system capable of remotely accessing a vehicle's subsystems through a GSM network. This system not only immobilizes the vehicle in case of theft but also enables users to track the vehicle's location. It provides users with the ability to remotely monitor their vehicles through the mobile network. The project employs Arduino, GSM, GPS, and Arduino Uno software with an ignition key.
LIBRARY MANAGEMENT SYSTEM Our library management system meticulously maintains records of books and individuals utilizing library services for reading and borrowing. It serves the needs of a wide range of users. The system includes an admin role responsible for customer record-keeping, while customers can access library services to borrow books, check their balance, and monitor the number of books they have borrowed. This project utilizes HTML, CSS, and JavaScript for the frontend and MySQL as the database
